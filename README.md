## رفع الموقع على GitHub Pages

### الطريقة الأولى: عبر واجهة GitHub

1. سجل الدخول إلى حسابك على GitHub
2. اضغط على "New repository"
3. أدخل اسم المشروع: `dallah-hospitality`
4. اختر "Public"
5. اضغط "Create repository"
6. اضغط "uploading an existing file"
7. ارفع ملف `index.html`
8. اضغط "Commit changes"
9. اذهب إلى Settings > Pages
10. في "Source" اختر "main" branch
11. اضغط "Save"
12. الموقع سيكون متاحاً على: `https://username.github.io/dallah-hospitality`

### الطريقة الثانية: عبر Git Command Line

```bash
# إنشاء مجلد المشروع
mkdir dallah-hospitality
cd dallah-hospitality

# نسخ ملف الموقع إلى المجلد

# تهيئة Git
git init
git add index.html
git commit -m "Initial commit - Dallah Hospitality Website"

# ربط مع GitHub
git remote add origin https://github.com/username/dallah-hospitality.git
git branch -M main
git push -u origin main
```

ثم فعّل GitHub Pages من إعدادات المشروع.
